A simple note taking app with multi user support that allows you to take notes and save them in a SQLite database.

## Features
- Users
  - Register
  - Login
  - Logout
  - Change password
  - Change display name
  - Password Hashing with salted SHA-512
- Notes
  - Create
  - Edit
  - Delete
  - Share
  - Search
  - Sort
  - Filter
